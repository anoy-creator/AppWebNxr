# AppWebNxr — Documentation technique

Documentation technique courte et pratique pour les contributeurs — voir runEnv.md pour la liste rapide des commandes.

## Présentation
Application Symfony conteneurisée (Docker, Nginx, MariaDB). Utiliser les commandes dans `runEnv.md` pour démarrer et pousser.

## Stack
- PHP 8.3 (FPM)
- Symfony
- Nginx
- MariaDB 11
- Composer
- Docker & Docker Compose

## Où trouver les commandes
Voir `Concept/technique/runEnv.md` (liste des commandes pour exécuter le projet localement et push Git).
